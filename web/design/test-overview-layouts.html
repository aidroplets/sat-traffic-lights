Tab style variations

All full-width. Different shapes, different active-state treatments, different relationships to the container above/below.

Tab T1

Pill row — bold active fill

Same shape as today but full-width and the active tab gets a solid lavender fill with contrasting text. Hard to miss which is selected.

Tab T2

Segmented control — iOS-style

Rectangular segmented control. Active gets a raised "card" look with subtle shadow + lighter bg. Reads like a hardware toggle.

Tab T3

Underline tabs (flush, no container)

No pill, no background. Each tab is flat text spanning half the row, with a thick underline marking the active one. Sits flush at the top of the parent — no padding above. Classic web-app dashboard pattern.

[Content here — e.g., focus list + CTA. The tabs are flush with the card's top edge.]
Tab T4

Split-fill — active half tinted

A single pill split in half. The active half fills with the tint color, the inactive stays transparent. Less heavy than T1's solid fill; cleaner than the underline.

Tab T5

Card-header tabs (active = card top)

The active tab IS the card header — its background extends down into the card. The inactive tab sits next to it like a recessed sibling. Strongest sense of "you are here".

[Content here uses the same bg as the active tab — they read as one continuous panel.]
Tab T6

Pill with bordered active

Full-width pill row but the active tab is marked by a brighter border + slightly elevated background. Subtler than T1's solid fill, more contained than T2's segmented look.

Tab T7

Big split — dramatic active state

Tall, wide tabs that double as section headers. Active tab gets a tinted background + indicator dot. Inactive is dimmed. High-contrast, very obvious.

Tab T8

Sliding-indicator pill (animated)

Pill row with a SLIDING active background that animates between tabs on click (try clicking). Most polished; the motion makes the change feel intentional and the selected state unmistakable.
